Dynamic color system

LS v5 comes with a completely reworked dynamic color system.
It still comes with the same accents as before, but managing colors is now much easier and more flexible.
The new color system also standardizes color naming and allows to create beautiful interfaces that respond to the scene.

Color themes can be generated either from the built-in presets, a static color, mix of colors or an image.
